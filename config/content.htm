W2

常用指令&語法

git指令 

//設定用戶名
git config --global user.name "your_name"
 
//設定用戶信箱位址
git config --global user.email "your_name@example.com"

//複製現有倉儲
git clone <倉儲網址>

//新增全部更改過的檔案至暫存區域
git add .

//將全部檔案從暫存區域中移除
git reset .

//提交並附上記錄之名稱
git commit -m "commit message"

//合併近端遠端儲存庫
git pull

//將暫存區域之檔案推送至遠端
git push

//擷取子目錄
git submodule add

pip指令

//安裝套件
pip install

//更新套件
pip install -U 

//移除套件
pip uninstall

//查詢安裝過之套件清單
pip list

HTML語法

//超連結
<a href="連結網址">連結名稱</a>

//普通分隔線
<hr>

//彩色分格線
<hr color=色號>

//文字、圖片置中
<center></center>

//文字色彩調整
<span color="色號" style="color:色號;">字</span>
